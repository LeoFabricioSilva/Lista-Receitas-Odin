# Projeto Lista de Receitas Odin

Este projeto consiste em um site simples de receitas culinárias.  
A página principal apresenta o nome e a imagem de cada prato, que direcionam para páginas individuais contendo as receitas.

## Tecnologias utilizadas

- HTML5
- Git
- GitHub

## Objetivo

Praticar conceitos básicos de HTML, estruturação de páginas e navegação entre arquivos utilizando links.
